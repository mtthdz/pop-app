# pop

Chat app built with [Expo](https://expo.dev).

## Get started

1. Install dependencies

   ```bash
   npm install
   ```
2. This project requires a few dotenv variables:

   ```bash
   EXPO_PUBLIC_SUPABASE_URL
   EXPO_PUBLIC_SUPABASE_ANON_KEY
   ```

3. Start the app

   ```bash
    npx expo start
   ```
